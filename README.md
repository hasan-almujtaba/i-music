# Introduction

Vue 3 Web with itunes api

## Live Demo

[Live Demo](https://i-music-core.netlify.app)

### Clone to local

```bash
pnpm dlx degit hasan-almujtaba/vue-starter my-app
cd my-app
pnpm i
```

## Usage

### Development

Start local environment

```bash
pnpm dev
```

### Build

To build the App, run

```bash
pnpm build
```
